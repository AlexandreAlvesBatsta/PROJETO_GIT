Teste de alteração
git init
git add "nome_arquivo"
git add . (todos modificados e adiciona)
git commit -m "Tutorial Git"
git status
git push (sincroniza tudo para o github)
git clone link (cria um rep. local cópia do rep. remoto)
git remote show origin (verifica se o rep local está atualizado com rep remoto)
git pull (atualiza rep local com as ultimas info do rep remoto)
git branch nome_branch (criar nova branch)
git checkout nome_branch (mudar para outra branch)
git merge nome_branch (junta a branch atual com a branch especificada no comando)
